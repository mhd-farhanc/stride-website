## About Stride

[Stride](https://github.com/mhd-farhanc/stride-app) is a beautiful, minimalist step tracker built with Flutter. It features:

- **Stark red, black & white theme** — bold, readable, intentional
- **Ring-based progress** — watch your daily goal fill as you move
- **7-day history chart** — review your weekly trends at a glance
- **Auto-reset daily** — every day is a fresh start
- **100% private** — all data stays on-device. Nothing uploaded, nothing shared.

---

## About This Site

This is the official marketing landing page for Stride. It's a single-page HTML site built to showcase the app's design philosophy, tech stack, and values — with zero framework overhead.

### Design Highlights

- **Dark, minimalist aesthetic** — pure black backgrounds with accent red
- **CSS-only phone mockup** — no images, no iframes
- **Floating glass tooltip** — frosted glass effect with `backdrop-filter`
- **Animated rotating badge** — SVG circular text badge with continuous spin
- **Fully responsive** — adapts from desktop to mobile seamlessly
- **Font system** — Anton for headings, Inter for body

## Getting Started

This is a static site — no build tools required.

```bash
# Clone the repo
git clone https://github.com/mhd-farhanc/stride-website.git
cd stride-website

# Open in your browser
open index.html  # or double-click in your file manager
```

## Deployment

The site is deployed via **GitHub Pages** from the `main` branch:

```bash
git push origin main
```

Enable GitHub Pages in your repo **Settings > Pages** and set the source to the `main` branch root.
